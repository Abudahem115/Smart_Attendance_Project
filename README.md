# Smart Attendance Project

A Face Recognition based attendance system integrated with Supabase.

## Features
- Face Registration & Recognition
- Duplicate Face Prevention
- Attendance Logging (Once per day)
- Admin Dashboard (Flask)
- Raspberry Pi Support

## Setup
1. Clone repository
2. Install dependencies: `pip install -r requirements.txt`
3. Setup `.env` with Supabase credentials.
4. Run server: `python server.py`
